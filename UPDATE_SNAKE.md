# Обновление игры "Змейка" на Windows

## 🔧 Что исправлено

**Проблема:** Игра загружалась, но змейка не двигалась (только реклама и пустое поле).

**Решение:** Теперь змейка автоматически начинает движение вправо при загрузке страницы.

---

## 📥 Скачать обновлённую версию

### Способ 1: Через PowerShell

```powershell
Invoke-WebRequest -Uri "https://raw.githubusercontent.com/Art-90362/snake-game-20260616/master/snake.html" -OutFile "$env:USERPROFILE\Downloads\snake.html"
```

### Способ 2: Через браузер

1. Откройте: https://github.com/Art-90362/snake-game-20260616
2. Нажмите на `snake.html`
3. Нажмите кнопку **«Raw»**
4. Правой кнопкой → **Сохранить как** → `snake.html`

---

## 🎮 Как играть (после обновления)

1. Откройте `snake.html` в браузере (Edge/Chrome/Firefox)
2. **Змейка начнёт двигаться автоматически!**
3. Управление:
   - **Стрелки** или **WASD** — движение
   - **Русская раскладка** (ЦФЫВ) — тоже работает
4. Съедайте красные яблоки → змейка растёт
5. Избегайте столкновений со стенами и своим хвостом

---

## 📊 Что изменилось в коде

```javascript
// Было:
let dx = 0;  // Змейка стояла
let dy = 0;

// Стало:
let dx = 1;  // Движение вправо
let dy = 0;
```

Теперь при загрузке игры змейка сразу начинает ползти вправо. Нажмите любую клавишу управления, чтобы изменить направление.
