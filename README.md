# Snake Game

Классическая игра Змейка на Python с использованием Pygame.

## 🎮 Как играть

- **Управление:** Стрелки или WASD (включая русскую раскладку)
- **Цель:** Съедай красные яблоки, чтобы змейка росла и набирала очки
- **Проигрыш:** Если врезаешься в стену или в свой хвост — игра окончена

## 📦 Установка

### Способ 1: Используй HTML-версию (рекомендуется!)

Просто открой `snake.html` в браузере — работает без установки Python!

```
http://mevm.art.it-dl.ru:8080/snake.html
```

Или открой файл в браузере через проводник.

### Способ 2: Python-версия

Если хочешь играть в терминале с управлением через стрелки:

1. Установи зависимости (нужен Python 3.7+):
```bash
pip install -r requirements.txt
```

2. Запусти игру:
```bash
python snake.py
```

**Примечание:** Для установки `pygame` могут потребоваться системные библиотеки SDL. Если возникают ошибки, используй HTML-версию!

## 🛠 Требования

- Python 3.7+ (для Python-версии)
- Pygame 2.0+ (для Python-версии)

## 📁 Структура проекта

```
snake/
├── snake.py          # Основной код игры (Python + Pygame)
├── snake.html        # HTML-версия (веб-игра, работает без Python!)
├── requirements.txt  # Зависимости (pygame)
├── setup.py          # Конфигурация установки
└── README.md         # Документация
```

## 🎨 Настройка

Можешь изменить:
- Размер поля (`gridSize`)
- Скорость игры (`FPS` в Python, `gameInterval` в HTML)
- Цвета (зелёная змейка, красная еда)
- Размер окна
